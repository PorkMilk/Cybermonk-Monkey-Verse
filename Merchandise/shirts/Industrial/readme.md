
# Industrial Brand Universe

## Purpose

The Industrial Brand Universe is a scalable design framework for creating print-on-demand apparel collections built around industrial classifications, symbols, administrative designations, and technical visual language.

All collections should follow the standards defined in [[Design Standards]].

# Design Formula

All collections should be built around the core design formula:

SYMBOL

CLASSIFICATION

OPTIONAL SUPPORTING TEXT

Examples:

- ☣ TOXIC
    
- ⚡ LETHAL
    
- ♻ RECLAIMED
    
- ✓ APPROVED
    
- ✕ REJECTED
    

The symbol and classification should carry the majority of the visual weight.
